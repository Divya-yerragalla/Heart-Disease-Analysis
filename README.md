# Heart Disease Analysis Using Tableau

## Project Overview
Heart disease is one of the leading causes of mortality worldwide, driven by factors such as unhealthy lifestyles, lack of physical activity, and underlying health conditions. This project focuses on analyzing heart disease data using **Tableau** to transform raw healthcare data into **interactive and meaningful visual insights** that support preventive healthcare and informed decision-making.

---

## Objectives
- Analyze heart disease trends across age, gender, race, and health conditions  
- Identify high-risk groups based on lifestyle factors such as smoking, alcohol consumption, BMI, and physical activity  
- Visualize correlations between heart disease, diabetes, and stroke  
- Build interactive Tableau dashboards and stories  
- Support healthcare professionals, policymakers, and individuals in preventive decision-making  

---

## Problem Statement
Traditional healthcare reports struggle to reveal complex patterns in large heart disease datasets. There is a need for **interactive data visualization** that simplifies complex information and enables faster, data-driven insights.
