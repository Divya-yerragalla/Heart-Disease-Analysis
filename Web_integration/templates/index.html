Heart Disease Analysis

This website presents a comprehensive analysis of heart disease data using interactive dashboards and visual stories. The goal of this project is to identify patterns and factors that influence heart disease risk.

About This Project

This project focuses on analyzing heart disease data to identify important health and lifestyle factors that contribute to heart disease risk. Using data visualization techniques, the project presents meaningful insights through interactive dashboards and visual stories. The analysis includes factors such as age, gender, smoking habits, physical activity, cholesterol levels, and diabetes. Tableau Public dashboards are used to clearly represent patterns and trends in the data. The project helps in understanding how different medical and lifestyle factors influence heart health. A web-based interface was developed using HTML, CSS, and Flask to present the dashboards and stories in an organized way. The main objective of this project is to improve awareness and support early understanding of heart disease risk factors.
