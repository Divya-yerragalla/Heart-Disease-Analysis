Cardiovascular Risk Analysis

A data-driven platform for cardiovascular risk assessment powered by advanced machine learning models. The system evaluates critical health, behavioral, and clinical indicators to generate accurate risk predictions and actionable insights. Designed to support early detection and preventive care, it enables users to make informed decisions about their heart health.

About This Project

This project presents an intelligent cardiovascular risk analysis system that integrates machine learning with interactive data visualization. The platform identifies and evaluates key health and lifestyle factors influencing heart disease through predictive modeling techniques. By leveraging structured healthcare data, the system analyzes parameters such as age, BMI, physical activity, medical history, and behavioral patterns to estimate risk levels with high reliability. The machine learning model is trained on real-world datasets to ensure meaningful predictions and consistent performance. The application is developed using HTML, CSS, and Flask, providing a clean and user-friendly interface for exploring insights, visual dashboards, and personalized risk assessments. The primary objective is to enhance awareness, support early diagnosis, and promote proactive healthcare decisions.
